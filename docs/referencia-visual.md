Crea una presentación HTML ejecutiva de resultados de marketing digital, tomando como referencia visual las imágenes ubicadas en:

assets/referencias/canva/

## Objetivo

Desarrollar una presentación moderna, limpia y profesional para mostrar resultados trimestrales de marketing digital de Quálitas.

La presentación debe sentirse similar a una presentación diseñada en Canva: visual, clara, con mucho aire, pocos textos por slide y capturas de pantalla como evidencia principal.

## Estilo visual

- Formato 16:9.
- Diseño ejecutivo, minimalista y moderno.
- Uso amplio de espacios en blanco.
- Jerarquía tipográfica muy clara.
- Títulos grandes y contundentes.
- Textos breves, fáciles de leer.
- Composiciones tipo:
  - Portada
  - Resumen ejecutivo
  - Métricas principales
  - Datos demográficos
  - Mensajes y contactos
  - Contenido destacado
  - Conclusión
  - Insights clave
  - Áreas de oportunidad
  - Siguientes pasos
  - Cierre

## Inspiración de diseño

Usa las imágenes de referencia solo como guía visual, no las copies literalmente.

Replica el estilo general:
- Fondos claros con degradados suaves.
- Grandes titulares alineados a la izquierda.
- Uso de etiquetas/píldoras para secciones como “Facebook”, “Resumen”, “Insight clave”.
- Tablas limpias con bordes redondeados.
- Tarjetas con capturas de pantalla.
- Gráficas simples y bien espaciadas.
- Elementos decorativos abstractos, sutiles y corporativos.
- Footer discreto con el nombre del reporte.
- Navegación visual sencilla con flechas.

## Paleta de color

Adaptar el estilo a la identidad de Quálitas:

- Amarillo Quálitas como acento principal.
- Negro para textos principales.
- Blanco para fondos.
- Gris claro para bloques secundarios.
- Gris oscuro para textos de apoyo.

Evitar saturar la presentación con amarillo. Usarlo solo para acentos, badges, líneas, destacados o pequeños elementos gráficos.

## Assets

Usar los archivos locales:

- Fondos: assets/fondos/
- Logos: assets/logos/
- Capturas: assets/capturas/
- Referencias visuales: assets/referencias/canva/

Las capturas deben integrarse como evidencia principal, dentro de marcos limpios, tarjetas o layouts amplios.

## Estructura sugerida

1. Portada  
   Título: Resultados de Marketing Digital  
   Cliente: Quálitas Compañía de Seguros  
   Periodo: [agregar periodo]  

2. Resumen ejecutivo  
   Breve lectura del trimestre + tabla comparativa de métricas clave.

3. Datos demográficos  
   Mostrar género, edad y ubicación principal de la audiencia.

4. Mensajes y contactos  
   Presentar contactos, conversaciones, respuesta y origen.

5. Contenido destacado  
   Integrar capturas de publicaciones con etiquetas como:
   - Educativo
   - Promocional
   - Conectivo

6. Conclusión general  
   Texto breve con lectura estratégica del trimestre.

7. Insights clave  
   4 insights principales en formato visual.

8. Áreas de oportunidad  
   Lista clara de oportunidades para mejorar conversión, pauta y contenido.

9. Siguientes pasos  
   Recomendaciones estratégicas para el próximo trimestre.

10. Cierre  
   Slide de agradecimiento con logo, redes y sitio web.

## Reglas de diseño

- No saturar los slides.
- No usar párrafos largos.
- Priorizar lectura ejecutiva.
- Mantener consistencia visual en todos los slides.
- Usar capturas como evidencia, no como decoración.
- Cada slide debe comunicar una sola idea principal.
- Evitar layouts demasiado cargados.
- Mantener márgenes amplios y buena respiración visual.

## Entregable

Genera una presentación HTML funcional con:

- index.html
- styles.css
- script.js si es necesario

Debe poder abrirse localmente en el navegador.

Agregar navegación entre slides con teclado y botones/flechas.

El resultado debe sentirse como una presentación ejecutiva personalizada, no como una página web genérica.
